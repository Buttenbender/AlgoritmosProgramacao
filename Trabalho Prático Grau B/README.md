## Trabalho Prático do Grau B: Sistema de Gerenciamento de Poções de Alquimia
# Autor
 - João Pedro Büttenbender da Silva
# Descrição
```
Este programa é um simulador de preparação de poções mágicas que utiliza uma interface de
menu simples. Ele permite ao usuário gerenciar ingredientes e criar poções de acordo com receitas
pré-definidas. O programa é estruturado com três funcionalidades principais:
1 - Consultar Ingredientes Disponíveis
2 - Preparar Poções
3 - Reabastecer Ingredientes
```
